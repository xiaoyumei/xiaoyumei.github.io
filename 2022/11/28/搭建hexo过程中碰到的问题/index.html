搭建hexo过程中碰到的问题

发表于2022-11-28|更新于2022-11-28
|阅读量:

1、github 创建page没有生成链接地址，原因是仓库命名没有与账号名称一致导致
2、hexo安装主题时，需要先安装相应依赖，例如：
npm install hexo-renderer-pug hexo-renderer-stylus –save
3、当前使用的是butterfly主题，备案号悬挂问题，需要修改主题footer文件，对应的使用文档链接：https://butterfly.js.org/posts/21cfbf15/
4、新建写作需要命令行操作（具体内容编辑还需要下载相关编辑器，用起来好像没那么方便，有可能是刚开始接触，不太会用）
hexo create
